① Raw Data
1
FROM
Table load hoti hai, JOINs
✅ Table columns ✅ JOINs / subqueries
Starting point — sab kuch yahan se aata hai
2
WHERE
Raw rows filter karna
✅ Column values ❌ SELECT aliases ❌ SUM / COUNT / AVG
WHERE SUM(amount) > 0 — galat! → HAVING use karo
② Grouping
3
GROUP BY
Rows ko groups mein todna
✅ Column names ❌ SELECT aliases ❌ Aggregate functions
Sirf real column names — aliases abhi bane nahi
4
HAVING
Groups filter karna
✅ SUM / COUNT / AVG ✅ Column names ❌ SELECT aliases
WHERE aggregates nahi le sakta tha — HAVING le sakta hai
③ Output
5
SELECT
Columns + aliases define hote hain yahan
✅ Columns, aggregates, expressions ✅ Aliases — pehli baar yahan bante hain
Isliye WHERE / GROUP BY / HAVING aliases nahi jaante the
6
WINDOW (OVER)
Window functions — GROUP BY ke baad chalti hain
✅ Grouped result ✅ sum(sum(x)) — nested aggregates
GROUP BY already ho chuka → isliye SUM ke upar SUM chal sakta hai
④ Final
7
ORDER BY
Sort — SELECT aliases use ho sakte hain ✨
✅ Columns, aggregates ✅ SELECT aliases ← sirf yahan exception!
SELECT ke baad execute hota hai → aliases pata hain
8
LIMIT
Final rows cut karna
✅ Final sorted result
Sab kuch ho chuka — phir rows count limit hoti hain
